# Erode Mandi Price Volatility & Market Opportunity Dashboard

A Power BI mini project analyzing mandi prices in **Erode district (2021–2025)** to identify **price trends, volatility, premium markets, and better selling opportunities**.

## Project Overview
This dashboard transforms raw agricultural market data into actionable insights for comparing commodities, markets, and seasonal price movement.

## Tools Used
- Power BI
- Power Query
- Excel
- DAX

## Key Features
- Interactive dashboard with slicers for commodity, market, year, price band, and spread category
- Executive overview of price levels and market coverage
- Volatility analysis using price spread
- Market opportunity analysis using price, spread, and recent movement
- Time-based analysis with monthly, quarterly, MoM, and YoY comparisons

## Dashboard Pages
- **Executive Overview** – overall market trends, top commodities, top markets
- **Volatility Analysis** – most volatile commodities and markets
- **Market Opportunity** – top opportunity commodities and markets

## Key Insights
- Covered **104 commodities** across **26 markets**
- Average modal price: **₹4.82K**
- Average price spread: **₹650**
- Premium commodities: **Cashewnuts, Jasmine, Kakada**
- Premium markets: **Mylampadi, Sivagiri, Moolanur**
- Most volatile commodity: **Dry Chillies**
- Most volatile market: **Moolanur**
